3.1.3 讨论
实际上，这种解压赋值可以用在任何可迭代对象上面，而不仅仅是列表或者元组。
包括字符串，文件对象，迭代器和生成器。
代码示例：
>>> s='Hello'
>>> a, b, c, d, e =s
>>> a
'H'
>>> b
'e'
>>> e
'o'
>>>
有时候，你可能只想解压一部分，丢弃其他的值。对于这种情况 Python 并没有提
供特殊的语法。但是你可以使用任意变量名去占位，到时候丢掉这些变量就行了。
代码示例：
>>> data =['ACME',50,91.1, (2012,12,21) ]
>>> _, shares, price, _ =data
>>> shares
50
>>> price
91.1
>>>
你必须保证你选用的那些占位变量名在其他地方没被使用到。
