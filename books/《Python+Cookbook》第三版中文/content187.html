8.3.3 讨论
在很多应用程序中处理 XML 编码格式的数据是很常见的。不仅因为 XML 在
Internet 上面已经被广泛应用于数据交换，同时它也是一种存储应用程序数据的常用格
式(比如字处理，音乐库等)。接下来的讨论会先假定读者已经对 XML 基础比较熟悉
了。
