7.3 5.3 使用其他分隔符或行终止符打印
7.3.1 问题
你想使用 print() 函数输出数据，但是想改变默认的分隔符或者行尾符。
7.3.2 解决方案
可以使用在 print() 函数中使用 sep 和end 关键字参数，以你想要的方式输出。
比如：
>>> print('ACME',50,91.5)
ACME 50 91.5
>>> print('ACME',50,91.5, sep=',')
ACME,50,91.5
>>> print('ACME',50,91.5, sep=',', end='!!\n')
ACME,50,91.5!!
>>>
使用 end 参数也可以在输出中禁止换行。比如：
>>> for iin range(5):
... print(i)
...
0
1
2
3
4
>>> for iin range(5):
... print(i, end=' ')
...
01234>>>
