使用 itertools.combinations() 可得到输入集合中元素的所有的组合。比如：
>>> from itertools import combinations
>>> for cin combinations(items, 3):
... print(c)
...
('a','b','c')
>>> for cin combinations(items, 2):
... print(c)
...
('a','b')
('a','c')
('b','c')
>>> for cin combinations(items, 1):
... print(c)
...
('a',)
('b',)
('c',)
>>>
对于 combinations() 来讲，元素的顺序已经不重要了。也就是说，组合 ('a',
'b') 跟('b', 'a') 其实是一样的 (最终只会输出其中一个)。
在 计 算 组 合 的 时 候， 一 旦 元 素 被 选 取 就 会 从 候 选 中 剔 除 掉 (比 如
如 果 元 素’a’ 已经被选取了，那么接下来就不会再考虑它了)。 而 函 数
itertools.combinations with replacement() 允许同一个元素被选择多次，比如：
>>> for cin combinations_with_replacement(items, 3):
... print(c)
...
('a','a','a')
('a','a','b')
('a','a','c')
('a','b','b')
('a','b','c')
('a','c','c')
('b','b','b')
('b','b','c')
('b','c','c')
('c','c','c')
>>>
