python-CGI编程

Date 2016-10-09（周日） Category python Tags python / cgi
  • 找到下面行：
AddHandler cgi-script .cgi

去掉前面的#，在后面加上.py,如下：

AddHandler cgi-script .cgi .py
  • 找到下面行：
 Options +Indexes +FollowSymLinks

在后面加上+ExecCGI,如下：

Options +Indexes +FollowSymLinks +Multiviews +ExecCGI
  • 保存，重启apache。
  • 在index.py文件中添加如下：
